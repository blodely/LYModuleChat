### Installation

LYModuleChat is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
# for now, using:
pod 'LYModuleChat', :git => 'https://github.com/blodely/LYModuleChat'

# Release:
pod 'LYModuleChat'
```
